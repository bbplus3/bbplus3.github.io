05 Time Series · Anomaly Detection

Utility Bill Forecasting & Anomaly Detection

A Python Shiny application that forecasts household utility bills using Facebook Prophet with custom-tuned seasonality parameters for three utilities (electricity, water, garbage), each on different billing cycles. Five anomaly detection algorithms run simultaneously — Z-Score, Z-Score on max, Local Outlier Factor, Isolation Forest, and a neural autoencoder — and are ensembled into a single combined anomaly score.

"Hyperparameters were tuned via cross-validation and documented inline — the code explains the tradeoffs between minimizing MSE and preserving interpretable seasonality."
Python ShinyProphet sklearn MLP AutoencoderIsolation Forest LOFAutoencoder SARIMAXStatsmodels
